Adding new map

To capture all boundary changes we sometimes need to refer to historical maps showing the boundaries of the provinces prior to the change.

Instructions:
  1. Here are some useful sources for finding historical maps:
    • MapWarper
    • OldMapsOnline
    • The Library of Congress Map Collection
    • The Perry-Castaneda Library Map Collection
    • Alabama Maps Historical Maps
    • World History at KMLA
    • Antique Maps and Prints
    • La bibliotheque du Defap

  2. Once you have found a map, first register the source where you found it, e.g. a book or a website. Make sure the map source doesn't have a license that restricts sharing or derivative work.

  3. Then register the map itself below, referencing the source you registered earlier.

  4. Finally, georeference the map at MapWarper and get its "WMS" link (under the "Export" tab). This will allow us to use the historical map as overlay when drawing boundary changes.
